# Что реализовано в RusCrypto Mini App (статус на сегодня)

Документ для команды ведения канала — что приложение умеет прямо сейчас, какие данные оно использует и по каким правилам считает свои оценки. Без технических деталей кода, только суть.

---

## 1. Что это за приложение

Telegram Mini App для криптоаудитории. Внутри: главный экран со сводкой по рынку, чек-лист оценки токенов перед входом, риск-календарь макро- и крипто-событий, индикатор уровня BTC с торговыми зонами, портфельный калькулятор и трекер, словарь терминов, гайды, бонусы от партнёров (биржа, виртуальная карта). Часть функций бесплатна, часть — платная через Telegram Stars.

---

## 2. Разделы приложения

| Раздел | Что внутри |
|---|---|
| **Главная** | Сводка: статус рынка, уровень BTC, ближайшее важное риск-событие. Всё подгружается из общего быстрого кэша. |
| **Чек-лист токена** | Пользователь выбирает токен → получает оценку риска входа (0–100 баллов) с вердиктом и пояснением. |
| **Обзор токенов** | Список токенов с базовыми рыночными данными. |
| **Уровень BTC** | Индикатор, показывающий, "перегрет" ли биткоин сейчас, и ближайшие ключевые уровни поддержки/сопротивления с рекомендацией (ждать/входить частями/не догонять). |
| **Риск-календарь** | Календарь на 7 дней вперёд: макроэкономические релизы (CPI, FOMC, NFP и т.п.) и анлоки токенов. |
| **Портфель → Калькулятор** | Просто делит введённую сумму по категориям риска в фиксированных пропорциях. Это НЕ трекер, реальных цен не использует. |
| **Портфель → Дневник** | Настоящий трекер: пользователь сохраняет свои реальные позиции, они хранятся в базе. |
| **Портфель → Готовый портфель** | Модельный долгосрочный портфель (статья/гайд), оформленный как готовый отчёт. |
| **Словарь терминов** | Статический справочник крипто-терминов. |
| **Гайды** | Обучающие материалы. |
| **Бонусы** | Партнёрка с биржей MEXC (бонус до 8000 USDT) — реклама, не функциональность приложения. |
| **Виртуальная карта** | Партнёрка Wanttopay — промо, не банковский продукт. |
| **Ещё / Админка** | Доп. меню; у админов — ссылка на внутреннюю аналитику использования приложения. |

## 3. Откуда берутся данные

- **CoinGecko** — цены, капитализация, объёмы для всех токенов в приложении (главный источник рыночных данных). Кэш на 1–60 минут в зависимости от раздела.
- **Котировки BTC для индикатора уровня** — берутся напрямую с бирж: сначала пробуется Binance, если не отвечает — Coinbase, потом Kraken. Кэш 15 минут.
- **Анлоки токенов** — агрегируются из нескольких источников по приоритету: Tokenomist → ряд точных провайдеров → CoinGlass → CoinMarketCal → запасной расчёт через CoinGecko → Mobula → Messari → CryptoRank. Если ни один источник не дал данных — показывается пометка "проверить вручную". Для BTC и ETH анлоков по определению нет — это правило встроено явно.
- **Макроэкономические события (CPI, PPI, ставка ФРС, NFP и т.п.)** — из FRED (официальный американский источник экономической статистики), плюс резервные источники Trading Economics, FMP, AlphaVantage.
- ⚠️ **Важно:** часть источников риск-календаря по факту **выключена**: парсер BLS не работает (блокируется сайтом), Fed/Treasury отключены, так как их данные и так покрываются через FRED. Также раздел крипто-новостей через Messari в риск-календаре **не реализован** — это заглушка, помечена в коде как "сделать позже".
- **Все данные о пользователях, платежах, позициях портфеля** — хранятся в собственной базе (Supabase).

## 4. Как считаются оценки (алгоритмы)

### Чек-лист токена (главная платная фича)
Стартовый балл — **70 из 100**. Дальше балл двигается по нескольким факторам:
- Перекупленность/перепроданность по RSI (от +8 за перепроданность до −30 за сильную перекупленность),
- Близость цены к историческому хаю/лою,
- Объём торгов относительно капитализации,
- Ликвидность,
- Общий риск рынка/BTC на этот момент (от −2 до −12),
- Доля токенов, ещё не выпущенных в оборот (от +2 до −15),
- Размер ближайшего анлок-события (от −3 до −15 в зависимости от того, насколько он крупный и близкий),
- Если данных не хватает — минус 15 баллов как штраф за неопределённость.

Итог переводится в вердикт:
- **до 40** — высокий риск, входить некомфортно;
- **40–59** — риск выше среднего, лучше не спешить;
- **60–74** — средний риск, можно изучать без спешки;
- **75 и выше** — риск низкий, зона комфортнее для входа.

### Индикатор уровня BTC
Самый сложный модуль. Кратко: система собирает котировки с бирж, считает технические индикаторы (RSI, скользящие средние, ATR), ищет на графике зоны поддержки/сопротивления (бывшие хаи/лои, круглые психологические уровни типа $80 000/$90 000/$100 000, средние линии). Каждая зона получает балл за количество касаний и объём — отсюда статус зоны: **ключевая / сильная / рабочая / слабая**.

"Перегрев" рынка фиксируется, если совпадают минимум 2 из 4 признаков: цена сильно выше своей средней, RSI выше 70, резкий рост за неделю, или цена впритык к сопротивлению.

На основе всего этого выдаётся одна из готовых рекомендаций пользователю: "можно докупать частями", "не догонять рост", "ждать возврата в диапазон", "ждать подтверждения" и т.п.

Есть один зашитый вручную уровень — сопротивление **$80 000–82 000** — добавлен как ручная поправка к автоматическому расчёту.

### Риск-календарь
Следит за 19 ключевыми активами. Если в ближайшие 48 часов есть важное событие — оно выводится на главный экран как приоритетное. Макрособытия делятся по важности: **высокая** (CPI, ставка ФРС, безработица, ВВП) и **средняя** (PPI, розничные продажи, PMI и т.п.) — по ключевым словам в названии релиза. Анлоки токенов делятся по доле от капитализации: **>2% — высокий риск**, **0.5–2% — средний**, меньше — низкий.

### Дневник портфеля (модельный)
Есть встроенная модель распределения по 16 активам с целевыми долями: ядро портфеля (BTC, ETH, SOL, BNB, LINK — 67% суммарно), спутник альфа (AAVE, HYPE, ONDO, SUI, TAO, UNI — 25%), спутник бета (AVAX, NEAR, JUP, PENDLE, RENDER — 8%).

## 5. Монетизация (Telegram Stars)

- **Бесплатно навсегда**: чек-лист по BTC и ETH.
- **Платно**: чек-лист по остальным 26 токенам (BNB, SOL, XRP, AAVE и др.) — **5 Stars за одну проверку** или **20 Stars за пакет из 5 проверок**.
- Повторный просмотр уже купленной проверки в течение **24 часов** не списывает деньги повторно.
- **Portfolio Pro** — подписка на 7 дней за **100 Stars**, продление добавляет дни к текущему сроку, а не считает с нуля.
- У администраторов доступ бесплатный по умолчанию (привязано к конкретному Telegram ID).

## 6. Что сейчас выглядит как заглушка / не доделано

- Статус рынка на главном экране — это **статичный текст, который обновляется руками**, а не живой расчёт.
- Страница "link-test" — техническая тестовая страница, в коде помечена как временная, к удалению.
